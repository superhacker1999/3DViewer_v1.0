Инструкция по использованию 3D Viewer

Установка:

  1. Вызвать цель "make install" из папки с  source-файлами.
  2. Дождаться установки *.app.

Удаление:

  1. Вызвать цель "make uninstall".
  2. Дождаться удаления  *.app.

Запуск и использование:

  1. Для запуска приложения открыть исполняемый файл 3D_viewer.app, либо через терминал командой "open 3D_viewer.app".
  2. Для начала работы загружаем 3D модель, используя кнопку "Выбрать файл", файлы моделей хранятся в папке datasets.
  3. Далее выбираем фон на котором будет отображаться модель нажатием на кнопку "Выбрать цвет фона".
  4. Также можно изменить цвет отображения ребер модели нажатием на кнопку "выбрать цвет" и в всплывающем окне выбрать цвет. Аналогично можно задать цвет для вершин модели.
  5. Далее можно установить толщину и тип для ребер и размер и вид для вершин.
  6. Для пермещения 3D модели вокруг оси X, Y или  Z, используем кнопки перемещения "x--" , "y--", "z--" или "x++" , "y++", "z++" соответственно.
  7. Для вращения модели используем аналогично ипользуем кнопки "x--" , "y--", "z--" или "x++" , "y++", "z++".
  8. Для масштабирования используем кнопку "%--" , чтобы отдалить модели и "%++" чтобы приблизить.

Запуск тестов:

  1. Для запуска тестов подсчета вызвать цель "make tests".
  2. Для создания html страницы с данными о покрытии функций тестами: вызвать цель "make gcov_report".
